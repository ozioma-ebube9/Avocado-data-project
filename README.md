# Avocado-data-project
Exploratory data analysis, preprocessing, dimensionality reduction using PCA, and machine learning on the Avocado dataset using Python.
Avocado PCA Analysis
Overview
This project explores the Avocado dataset using Python. The analysis includes data cleaning, feature engineering, dimensionality reduction using Principal Component Analysis (PCA), visualization, and machine learning classification.

Python
Pandas
NumPy
Matplotlib
Scikit-learn
Plotnine
Techniques Applied: 
Data Cleaning
Feature Engineering
Principal Component Analysis (PCA)
Classification Modeling
Model Evaluation
Data Visualization
